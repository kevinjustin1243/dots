# Kevin's Dots

Contains:
- colors from wallpaper with pywal16
- `.config` files

Software Requirements:
- `hyprland`
- `foot`
- `hyprpaper`
- `wofi`
- `pipewire`, `pipewire-pulse`, `wireplumber`

Directory Layout:
- Colors: has colorschemes for wallpapers, only Sakura right now
- Configs: has folders and files that would be found in a `.config` directory
- Wallpapers: has wallpapers used which would be placed in `~/Pictures`

Schemes:
- Sakura: wallpaper - `sakura.jpg`, colors - `Sakura`
